# AutoWhitelist
โปรเจ็คสำหรับ Server Minecraft ที่ต้องการทำ AutoWhitelist ด้วย SlipVerifyAPI

## Require
- NodeJS 18.X^
- MariaDB
- SlipVerifyApi: https://easyslip.com/
